### 3 | Set Environment Variables

| Variable name                 | Description                    |
| ----------------------------- | ------------------------------ |
| `REACT_APP_OPEN_WEATHER_API_KEY`   | OpenWeather API key       |
| `REACT_APP_GEONAMES_USERNAME` | Username for GeoNames API      |

```bash

```

```bash

npm start
```
